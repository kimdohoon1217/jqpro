:checked

- $('input[type=checkbox]:checked')
- $(':checkbox:checked')
- 선택한 요소를 대상으로 반복문 실행
- 반복문 안에서 현재 실행대상이 되는 객체를 this로 받는다
취 미 : 여행 장기 바둑 독서 낚시

:selected

- select/option에서 선택한 요소를 선택한다.
- select에서 change이벤트를 수행
- 선택한 요소를 대상으로 반복문 실행
- 반복문 안에서 현재 실행대상이 되는 객체를 this로 받는다
- multiple은 여러값을 가지게할수있는 속성
