dom객체와 jQuery 객체 변환

jquery를 dom으로 변환 : jQuery객체.get(N) / jQuery객체[N]
dom을 jquery로 변환 : $(dom객체)

jq = $('h1');
jq.css('color', 'blue');

- jQuery객체를 dom으로
dom = jq.get(2); //jq[2]
dom.style.color = "red";

- dom객체를 jQuery로 변환
$(dom).css('color', 'green');

- 전체 h1중 3번째 선택 : jQuery
jq3 = $('h1:eq(3)');
jq3.css('color', 'red');
